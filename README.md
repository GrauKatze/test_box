# TestBox
---
Утилиты для тестирования входных и выходных данных программы на предмет ее стабильности и соответствия выходных данных эталонным.

## Принцип работы

**TestBox** принимает в качестве аргументов путь до тестируемой программы, после чего запускает ее подавая заранее подготовленные входные данные, а так же сравнивает выходные с эталонными.

## Пример

Из комадной строки:
```bash
    test_box -p path/to/test_program
```
Используя Cargo
```bash
    cargo -r -- -p path/to/test_program
```

## Структура

Содержит модули:
- *test_box* - основной модуль для управления всей утилитой;
- *configurator* - настраивает рабоы программы и разюирает аргументы при запуске;
- *program* - модуль описывающий и управляющий непосредсвтенно целевой программой;
